# **Peek ORM**

---

## ✅ Scripts

**Clone the repo**

```bash
git clone https://github.com/thutasann/peek-orm.git
```

**Install Packages**

```bash
yarn
```

**Compile and build the lib**

```bash
yarn build
```

**Test**

```bash
yarn test
```

**Benchmark**

```bash
yarn benchmark
```

**Documentation Site**

```bash
yarn docs
```

**Examples Codes**

```bash
cd examples && yarn dev
```

---

## 📦 **Installation**

You can install **Peek ORM** via npm. Simply run the following command:

```bash
npm install peek-orm@latest
```
